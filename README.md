The configuration that lets me work, rather than procrastinate modifying it.

## Introducrtion
This <i>NeoVim</i> config was created as a result of being curious and in love with
idea of programming <strong>your own editor config</strong> with the only features
you ever needed.

> [!NOTE]
> Nerd font required

## Main Features
- LSP Completion: [blink.cmp](https://github.com/saghen/blink.cmp)
- Flexible formatting: [conform.nvim](https://github.com/stevearc/conform.nvim)
- LazyGit integration: [snacks.nvim](https://github.com/folke/snacks.nvim)
- Fuzzy finder: [fzf.lua](https://github.com/ibhagwan/fzf-lua)
- Soft & low-contrast colorscheme: [zenbones.nvim](https://github.com/zenbones-theme/zenbones.nvim)
